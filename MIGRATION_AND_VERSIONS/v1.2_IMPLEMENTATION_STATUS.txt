INSTALLATION & DEPLOYMENT INSTRUCTIONS
================================================================================

For Users (Standard Installation):
  1. Extract v1.2 package to local folder
  2. Read README.md (overview)
  3. Edit config.txt with customizations
  4. Run INSTALL.ps1 as Administrator
  5. Log out and back in to see welcome message
  6. Run VERIFY.ps1 to confirm installation

For Developers/Maintainers:
  1. Review BUG_FIXES_v1.2.md for all changes
  2. Check git history for detailed commits
  3. Review MIGRATION_GUIDE.md for upgrade paths
  4. Use VERIFY.ps1 for diagnostics
  5. Use CONFIG_VALIDATOR.ps1 for testing configurations

Troubleshooting:
  1. Run VERIFY.ps1 for diagnostic report
  2. Run CONFIG_VALIDATOR.ps1 to validate configuration
  3. Check FAQ.md for common issues
  4. Use UNINSTALL.ps1 -WhatIf to preview removal
  5. Check Windows Event Viewer for scheduled task failures
